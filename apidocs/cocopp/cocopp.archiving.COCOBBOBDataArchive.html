To see the list of all data from 2009:

>>> import cocopp
>>> cocopp.archives.bbob.find("2009/")  # doctest:+ELLIPSIS,+SKIP,

To use the above list in main:

>>> cocopp.main(cocopp.archives.bbob.get_all("2009/")  # doctest:+SKIP
